# AstroLog

An astronomical observation logging Flutter application with a beautiful dark space theme.

## Features

- **Home**: Welcome screen and overview
- **Explore**: Discover celestial objects and phenomena
- **Track**: Monitor your astronomical tracking and observations
- **Events**: View upcoming astronomical events
- **Profile**: Manage your observations and settings

## Project Structure

```
lib/
├── main.dart              # Entry point with bottom navigation
├── models/                # Data models
├── screens/               # Screen widgets (Home, Explore, Track, Events, Profile)
├── widgets/               # Reusable custom widgets
├── services/              # Business logic and API services
├── utils/                 # Helper functions and utilities
└── themes/                # App theme (dark space theme)
```

## Theme

The app features a dark space theme with:
- Deep navy background (#0A0E27)
- Space purple surfaces (#1A1A3E)
- Accent blue highlights (#4A6FE3)
- Custom text styles for headings and body text

## Getting Started

1. Make sure you have Flutter installed
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app

## Requirements

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
